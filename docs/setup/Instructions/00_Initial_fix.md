

# Fix following error

1. Host PIN and Guest PIN กินพื้นที่หน้าจอไปเกินครึ่งให้เอาออกแล้วย้ายไปที่ Menu Players
2. Menu Players ให้แสดง Host PIN and Guest PIN
3. Host role สามารถ add Player และระบุ Role ให้กับ Players ได้
4. Host ต้องไม่ Lock-out ตัวเองด้วยการ toggle ตัวเองจาก Host กลายเป็น Guest
5. Guest ดูรายชื่อ หรือแก้ไขชื่อตัวเองได้เพียงอย่างเดียว : Add, Toggle, Remove ตัวเองออกไม่ได้
6. Guest ถ้า Join ด้วยชื่อที่มีอยู่แล้ว ให้ Join เข้าไปเป็นคนนั้นได้เลย (ไม่ต้องสร้าง User ใหม่ )
